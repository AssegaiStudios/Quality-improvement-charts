# Statistical Reference

pyqicharts implements practical SPC calculations for run, Individuals, Moving Range, C, P, U, Xbar, S, G, T, P-prime and U-prime charts.

v1.1 adds Nelson Rules 1-8 and a Shewhart-compatible rule helper. These helpers return a stable signal schema so charting, Excel, Power BI and tests can use the same rule metadata.

The current validation fixtures are internal regression checks. External parity certification against qicharts, qicharts2, NHS examples and published SPC references remains tracked in `PARITY_REPORT.md`.
