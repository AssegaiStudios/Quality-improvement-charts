# pyqicharts v0.5.0

This release adds baseline periods, recalculation segments, targets, interventions and step-change metadata.

## Added

- `baseline_points`
- `recalculation_points`
- `target`
- `interventions`
- `step_changes`
- Segment metadata in chart tables
- Plotting support for target and vertical process markers

## Install

```bash
pip install pyqicharts
```

For local development from this source archive:

```bash
pip install -e .[dev]
pytest
```
