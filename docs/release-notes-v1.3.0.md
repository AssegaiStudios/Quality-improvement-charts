# pyqicharts v1.3.0 Release Notes

v1.3.0 adds the pyqicharts Excel Companion Phase 1 and Phase 2 implementation.

## Highlights

- New `pyqicharts_excel` companion package.
- Excel workbook templates with required Data, Config, output, Log and Help sheets.
- xlwings-callable runner functions for chart generation, validation, clearing outputs and report bundle export.
- Optional `[excel]` dependency group.
- `pyqicharts-excel-init` command for creating a fresh template.
- Excel setup, user guide, troubleshooting and completion report documentation.
