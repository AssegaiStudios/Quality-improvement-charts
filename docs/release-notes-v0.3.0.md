# Release notes: v0.3.0

## Highlights

Version 0.3.0 expands pyqicharts with commonly used attribute charts and the first visual theme system.

## Added

- C charts for count data
- P charts for proportion data
- U charts for rate data
- Dynamic centre-line labels
- Built-in themes: default, nhs, publication and dark
- NHS-inspired visual theme
- Theme support for Pareto charts
- Expanded table outputs
- Additional examples and tests

## Known limitations

- Anhøj thresholds still use an approximate implementation and need validation against published reference tables.
- Xbar, S, T, G, P′ and U′ charts are not yet implemented.
