# NHS Guide

pyqicharts supports NHS-style improvement interpretation with:

- `improvement="high is good"`
- `improvement="low is good"`

Individuals charts expose special-cause fields for points outside limits, shifts and trends. v1.1 adds a shared signal schema and Power BI signal/KPI tables to make NHS-style reporting easier to automate.

Full NHS Making Data Count workbook parity still requires external reference comparisons and is not claimed in this release.
