# pyqicharts v0.9.0

This release focuses on documentation, validation and release hardening.

## Added

- Documentation scaffold
- Validation datasets and expected outputs
- Example coverage for supported workflows
- GitHub Actions workflow
- Release hardening tests

## Install

```bash
pip install pyqicharts
pip install pyqicharts[reporting]
```

For local development from this source archive:

```bash
pip install -e .[dev]
pytest
```
