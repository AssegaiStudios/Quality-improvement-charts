# pyqicharts v0.9.0 Release Notes

pyqicharts v0.9.0 focuses on documentation, validation and release hardening.

## Highlights

- Added the roadmap documentation scaffold under `docs/`.
- Added deterministic validation datasets and expected outputs.
- Added release hardening tests for examples and validation outputs.
- Added GitHub Actions test workflow.
- Added runnable examples for supported chart families, reporting and Power BI.
- Added explicit placeholder examples for Xbar and S charts, which remain planned.

## Compatibility

Existing public APIs continue to work. v0.9 does not introduce major new chart calculations; it hardens the package for a future v1.0 release.

## Not Included Yet

v0.9 is not a v1.0 stability declaration. Expanded independent validation, full API documentation and compatibility/deprecation policy remain required before v1.0.
