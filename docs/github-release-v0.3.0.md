## New in v0.3.0

This release adds attribute charts and visual themes.

### Features

- C charts for count data
- P charts for proportion data
- U charts for rate data
- Dynamic centre-line labels
- NHS-inspired visual theme
- Publication and dark themes
- Improved signal highlighting
- Expanded Excel and Power BI table outputs
- Additional examples
- Expanded tests

### Notes

This is still an early alpha release. Please validate results before operational or production use.
